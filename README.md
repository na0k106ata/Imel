# Imel (IME Indicator)

**Imel (アイメル)** は、Windows 11 向けのシンプルなIME状態表示ツールです。
名前は「IME」と「見える（Mieru）」を組み合わせた造語で、入力状態が一目でわかるようにという意図が込められています。

マウスカーソルや文字入力カーソルのすぐそばに、現在の入力モード（「A」または「あ」）を表示します。視線をタスクバーに移動させることなく、手元で直感的に入力状態を確認できます。

## 特徴

* **直感的な表示**: マウスカーソルまたは文字入力カーソル（入力位置）に追従して表示します。
* **省リソース設計**: 常駐ツールとして邪魔にならないよう、メモリ使用量や描画負荷に配慮して設計されています。
* **高いカスタマイズ性**:
    * 文字色・背景色の変更（RGB指定可）
    * 背景の透過率調整
    * 表示位置の微調整
    * 描画更新頻度の調整（滑らかさ優先か、省エネ優先か）
* **スマートな挙動**:
    * 動画視聴時など、マウスカーソルが消えると自動的に表示も隠れます。
    * Windows起動時の自動実行（スタートアップ）に対応。
* **タスクトレイ常駐**: 作業を妨げず、いつでも設定画面へアクセス可能。

## 動作環境

* **OS**: Windows 11
* **ランタイム**: .NET 8.0 Desktop Runtime

## インストールと実行

### バイナリを使用する場合
リリースページから zip ファイルをダウンロードして展開し、`Imel.exe` を実行してください。

### ソースコードからビルドする場合
1.  このリポジトリをクローンします。
2.  Visual Studio 2022 で `Imel.csproj` を開きます。
3.  ソリューション構成を `Release` に設定し、ビルドします。
4.  `bin/Release/net8.0-windows` フォルダ内の `Imel.exe` を実行します。
