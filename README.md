# Imel (IME Indicator)

**Imel** は、Windows 11 向けの軽量なIME状態表示ツールです。
マウスカーソルやテキストカーソルのすぐそばに、現在の入力モード（「A」または「あ」）を表示します。視線をタスクバーに移動させることなく、直感的に入力状態を確認できます。

## 特徴

* **直感的な表示**: マウスカーソルまたはキャレット（入力位置）に追従して表示。
* **低負荷設計**: 徹底的なメモリ最適化と描画処理の効率化を行い、常駐時のメモリ使用量は極小です。
* **高いカスタマイズ性**:
    * 文字色・背景色の変更（RGB指定可）
    * 背景の透過率調整
    * 表示位置の微調整（オフセット）
    * 描画更新頻度の調整（滑らかさ vs 負荷）
* **スマートな挙動**:
    * 動画視聴時など、マウスカーソルが消えると自動的に表示も隠れます。
    * Windows起動時の自動実行（スタートアップ）に対応。
* **タスクトレイ常駐**: 邪魔にならず、いつでも設定画面へアクセス可能。

## 動作環境

* **OS**: Windows 11
* **ランタイム**: .NET 8.0 Desktop Runtime

## 開発環境

* Visual Studio 2022
* C# (WPF / .NET 8)
